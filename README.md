## 💧 WaterBot Zap

Criei esse projeto com o objetivo de acompanhar o consumo de água no dia a dia de forma simples e prática.

A ideia surgiu a partir da necessidade de controlar melhor minha hidratação, então desenvolvi um bot para WhatsApp que contabiliza o consumo em forma de pontos.

Cada pessoa pode definir como deseja contabilizar seu consumo:
- 1 ponto pode representar um copo
- uma garrafa
- ou até uma quantidade específica (como ml)

Optei por não fixar o valor em mililitros, pois o consumo de água varia de pessoa para pessoa. Nem todos utilizam recipientes do mesmo tamanho, seja em treinos ou no dia a dia.

Dessa forma, o sistema se adapta à realidade individual de cada usuário, permitindo um controle mais flexível e personalizado.

## 🚀 Funcionalidades

- Registrar consumo de água
- Sistema de pontos personalizado
- Histórico diário
- Consulta de consumo

## 🛠 Tecnologias

- Node.js
- whatsapp-web.js
- JavaScript
